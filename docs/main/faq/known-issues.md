# Known Issues & Crashes

<TerminalCard title="Common Bugs & Fixes">

### Crashes at launch
Usually caused by anti-virus blocking the launcher or active overlays (Discord, Steam, OBS). Add the Anomaly folder to your antivirus exclusions.

### Main menu delay
Wait a few moments for scripts to initialize. If it takes longer than 2 minutes, check for overlay conflicts.

### Stuck on black screen
If the game freezes on a black ENB screen, press **Alt+Tab** to cycle windows and return to the game.

### Stuck on main menu
This is often related to overlay conflicts or Buffout 4 initialization. Disable all overlays (Steam, Discord, etc.) and ensure you are using the provided save files if the "New Game" button is unresponsive.

### Falling through the vault
Do not enable the "Uneducated Shooter" optional mod until after you have left Vault 111.

</TerminalCard>
