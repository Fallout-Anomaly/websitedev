# Controls & UI

Fallen World features a highly customized interface designed to minimize clutter and maximize immersion. Many UI elements are dynamic and will only appear when relevant.

---

<TerminalCard title="Immersive Interface">

Our HUD configuration is designed to be as "dead" as possible until you need it.

<details>
<summary>View HUD Configuration & Controls</summary>

:::tip Key HUD Controls
*   **Toggle HUD**: Press **F2** to completely hide/show all on-screen elements.
*   **Dynamic Elements**: Health, ammo, and compass elements will auto-fade when you are out of combat or not interacting with the world.
*   **Customization**: Use the MCM to deeply customize which elements stay visible.
:::

</details>

</TerminalCard>

<TerminalCard title="Navigation & Menus">

Quickly access essential tools without fumbling through deep Pip-Boy menus.

<details>
<summary>View Fast-Access Menus</summary>

*   **Wheel Menu [H]**: A fast-access radial menu for consumable items and equipment.
*   **Quick Map [M]**: Instantly opens your local/world map.
*   **Quick Inventory [I]**: Opens a fast-access inventory screen.
*   **In-Game MODExplorer [ \ ]**: View detailed information about the active mods in your list.

</details>

</TerminalCard>
