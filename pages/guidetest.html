WHAT IS FALLOUT ANOMALY?

Fallout Anomaly is a true hardcore experience, bringing you as close to realism as possible with mods like Damn Apocalypse, MAIM, and more. Packed with new weapons and armor to face harsher challenges, Anomaly also features new enemy types, quests, settlement items, and a complete assortment of textures to elevate your game to modern standards.

DISCLAIMER
  • Beta Stages: Fallout Anomaly is currently in beta, meaning changes and evolution are inevitable.
  • Steam Version Only: At this time, only the Steam version of Fallout 4 is supported.
HARDWARE REQUIREMENTS
  • CPU: Modern CPU with at least 6 cores and 3GHz speed
  • RAM: 16GB minimum, 32GB ideal
  • GPU: At least 6GB VRAM, 8GB recommended
  • Storage: 350GB minimum on SSD (HDD not supported)
  • OS: Windows 10/11 (64-bit) with up-to-date drivers

🚨 Do not install Anomaly directly on a root drive (e.g., C:\Fallout Anomaly). Instead, install it within a dedicated folder structure, such as: C:\Fallout Modlist\Fallout Anomaly. 🚨

IMPORTANT NOTICE
  • 🚨 UPDATE FALLOUT 4 TO THE NEXT-GEN VERSION
  • 🚨 DISABLE ALL OVERLAYS (STEAM, MEDAL, ETC.)
  • 🚨 SET UP PAGE FILE FOR PERFORMANCE
  • 🚨 CONFIGURE ANTI-VIRUS EXCLUSIONS
IMPORTANT INSTALLATION STEPS
  1. Microsoft Visual C++ x64 and x86
    Must be installed! Open the launcher and follow the on-screen steps.
  2. NET Framework 4.6.2 & 5.0
    Required for the mod list to function properly.
  3. DirectX Redist (June 2010)
    Download and install to ensure proper DirectX support.
  4. Launch the Game Once
    Launch the game at least once to ensure it's ready for modding.
  5. PageFile Setup
    Critical for performance.
  6. Disable Overlays
    Disable all overlays to prevent crashes and performance issues.
PRE-INSTALLATION CHECKLIST
  • Log into Nexus in Wabbajack
  • Open Wabbajack, click the ⚙️ cogwheel, and log in under the Nexus panel
INSTALLATION INSTRUCTIONS
  • Locate Fallout Anomaly in the Wabbajack Browser and download the file
  • Set both Install Location and Download Location on the same drive
  • Click the play button to begin installation
COMMON ISSUES & FIXES
  • "Could Not Download MOD"
    • Manually download the mod and place it in Wabbajack's download folder
    • Check if Nexus Mods is down
  • "Mod Is Not a Whitelisted Download"
    • The modlist may have been updated
    • Wait for a new release or notify us on Discord
  • "Missing Game Files"
    • Ensure your game is updated to the next-gen version
ANTIVIRUS FOLDER EXCLUSION

To prevent interference, add a folder exclusion for your entire install directory:

  1. Open Windows Security → Virus & threat protection
  2. Scroll to Exclusions and select Add or remove exclusions
  3. Click Add an exclusion → choose Folder and browse to your install folder
UPDATING FALLOUT ANOMALY
  • Announcements on Discord and Nexus Mods
  • Re-run the installer on the same path and check "Overwrite existing"
  • Your save files remain intact unless we explicitly state otherwise
LAUNCHING FALLOUT ANOMALY

When you open Mod Organizer 2 (MO2), the interface might seem complex at first, but don't worry—we've streamlined the process!

  • In the top-right corner of the MO2 interface, click "Launch Fallout Anomaly" to start the game.

⚠️ DO NOT use the vanilla game launcher!

F4SE (FALLOUT 4 SCRIPT EXTENDER)

F4SE is essential for most Fallout mods and is required to run Fallout Anomaly.
When using MO2, F4SE is automatically included in "Launch Fallout Anomaly"—no extra setup needed!

CRASH HANDLING

If your game crashes, follow these steps to report the issue:

  1. Save the log to an easily accessible location.
  2. Upload the log via our bug report form OR post it in our Discord support section.
  3. Attach your most recent save file to help with debugging.
CHANGING RESOLUTION (BETHINI)

If your game's resolution isn't correct, use BethINI to adjust it. Do NOT modify other settings!

  1. Close Mod Organizer 2.
  2. Navigate to your Anomaly installation folder:
    • Open the Tools directory.
    • Inside, find and open the BethINI folder.
    • Right-click BethINI.exe and select "Run as Administrator".
  3. In BethINI, select Fallout 4 (DO NOT change INI paths!).
  4. Go to the Basic tab and adjust your resolution.
  5. Click Save & Exit to apply the changes.
APPLY THE MCM PRESET!

Don't forget to apply the preset for proper settings!

DISABLING BLACK BARS (LETTERBOX) IN ENB
  1. Press the ~ key (next to the 1 key) to open the console.
  2. Press END to open the ENB menu.
  3. Navigate to the Post Processing section.
  4. Uncheck the Letterbox option.
  5. Click Save Configuration (top-left).
  6. Press END and ~ again to exit the menus.
CONTROLLER SUPPORT

We currently support PS5 controllers, with Xbox support planned for the future.

  1. Ensure your Steam controller settings are correctly set up.
  2. Steam Overlay must be enabled for the controller to function properly.
  3. Steam Input: Install the custom controller profile for Fallout Anomaly.
  4. In MO2, activate the optional mod: "A - OPT - Controller Support - PS5 Support"
MAIN MENU DELAY

When launching the game you may experience a delay as scripts load. This delay varies based on your system's power; more powerful systems may not encounter this issue.

STUCK IN MENU

If you find yourself unable to progress from the first loading or main menu:

Watch this video for help with the issue. If the video doesn't resolve it, we've provided new save files for you to load and customize your character as needed.

POST-CHARACTER CREATION FREEZE
  • Your game will save and may appear to freeze temporarily.
  • This is due to multiple scripts loading simultaneously.
  • Be patient and wait for a few minutes; the game will resume normally.
FALLING THROUGH VAULT ISSUE

If you're falling through the vault floor:

  • This is likely due to the "Uneducated Shooter" mod.
  • Solution: Only enable the "Uneducated Shooter" mod after leaving the vault.
KNOWN CRASH RELATED ISSUES
  • Programs like medal (recording software) or anything that adds an overlay can conflict with the ENBS, or other DLL hooked mods.
  • If you are crashing right at the launch of game or at the main menu this can be one of the main causes.
SURVIVAL MECHANICS
  • Needs System: You must regularly eat, drink, and sleep to avoid penalties.
  • Fast Travel: Traditional fast travel is disabled. Use the APC Travel System or expanded subway system.
  • Saving the Game: Manual saving and quicksaving are disabled. Save by sitting in a chair or using a Survival Pack.
  • Combat Difficulty: Both you and enemies deal more damage. Healing is slowed.
  • Ammo Has Weight: Every bullet adds weight to your inventory.
  • Illness & Injury: You can get sick, suffer broken limbs, or require items for proper healing.
  • Carry Weight Reduced: Your base carry weight is significantly lower.
SKILLS AND LEVELS

Utilizes the YAE mod, which introduces new skills and modifies the perk system.

  • 26 total Skills
  • 250+ new and reworked Perks
  • More control over character growth
  • Choose a Background at the start for backstory, starter gear, and Skill bonuses
HEALING GUIDE

MAIM is a realistic combat and medical overhaul that introduces lethal headshots, injuries, bleeding, pain, and more.

  • SP-1 Stimpak: Gradual HP + AP regen, cures NPC injuries
  • SP-2 Stimpak: 3%/s HP & limb healing (40s)
  • Med-X: Treats pain, shock, and contusion
  • Bandage: Stops light bleeds, heals injured limbs
  • QuikClot Injector: Stops heavy bleeds
  • IFAK: Stops heavy bleed, removes crippled status
RADIATION SYSTEM
  • Dual-layered radiation system:
    • IRA (Ingested Radioactive Particles): From food, water, and ambient hazards
    • RAD (Tissue Damage): Traditional external radiation exposure
  • High IRA levels simulate Acute Radiation Syndrome
  • Weather hazards require protective gear
  • Protective gear is now breakable and must be maintained
COMBAT OVERHAUL

True Damage reworks Fallout's combat mechanics for realism and balance:

  • Realistic damage applied via ammo, scaled by caliber
  • Same-caliber weapons have logical damage ranges
  • Independent sliders for different weapon types
  • SCOURGE brings realism to NPC combat with randomized stat generation
KEYBIND PRESET

Fallout Anomaly includes custom keybinds for various functions:

  • Movement: WASD, Shift (Sprint), C (Run), Space (Jump), Ctrl (Sneak)
  • Combat: Mouse1 (Attack), Mouse2 (Aim), Alt (Bash/Power Attack), R (Reload), Q (VATS)
  • Menu: Tab (Pip-Boy), Esc (Pause), I (Quick Inventory), M (Quick Map)
  • Many mod-specific keybinds for enhanced features
