Installation Guide

Follow these steps to properly install and enjoy Fallout Anomaly

HARDWARE REQUIREMENTS
  • CPU: Modern CPU with at least 6 cores and 3GHz speed
  • RAM: 16GB minimum, 32GB ideal
  • GPU: At least 6GB VRAM, 8GB recommended
  • Storage: 350GB minimum on SSD (HDD not supported)
  • OS: Windows 10/11 (64-bit) with up-to-date drivers
  • Page File: Minimum 16GB required
  • Skills: Basic computer maintenance knowledge required
IMPORTANT NOTICE
  • Update Fallout 4 to the next-gen version
  • Disable all overlays (Steam, Medal, etc.)
  • Set up page file for performance
  • Configure anti-virus exclusions
  • Install in dedicated folder (not root drive)
  • Nexus Premium recommended for faster downloads
Download Required Components
  • Microsoft Visual C++ x64 and x86
  • NET Framework 4.6.2 & 5.0
  • DirectX Redist (June 2010)
  • English Voice Files (if not using English)
  • Verify all components are properly installed
Game Setup
  • Launch vanilla game at least once
  • Update to version 1.10.980.0
  • Remove HD texture pack if installed
  • Verify game files through Steam
  • Install in dedicated folder (not root drive)
  • Ensure clean game installation
System Configuration
  • Set up page file (minimum 16GB)
  • Configure anti-virus exclusions for game folder
  • Disable all overlays (Steam, Medal, etc.)
  • Update graphics drivers
  • Add exceptions for MO2.exe, Fallout4.exe, F4SE_Loader.exe
  • Add 'Stock Folder' from Mod Organizer 2 to exceptions
Wabbajack Setup
  • Log into Nexus in Wabbajack
  • Set Install Location and Download Location on same drive
  • Verify installation after completion
  • Check for errors in verification process
  • Contact Discord support if issues persist
Initial Launch
  • Use Mod Organizer 2 to launch the game
  • Do not use the vanilla launcher
  • F4SE is automatically included
  • Save crash logs if issues occur
  • Upload logs via bug report form
  • Attach recent save files for debugging
Resolution Settings
  • Use BethINI to adjust resolution
  • Do not modify other settings
  • Close Mod Organizer 2 first
  • Run BethINI as Administrator
  • Select Fallout 4 in BethINI
  • Do not change INI paths
Main Menu Delay
  • Initial delay as scripts load
  • Varies based on system power
  • More powerful systems may not experience delay
Crash Issues
  • Programs with overlays can cause conflicts
  • Disable Steam, Nvidia, Medal overlays
  • Conflicts with ENB and DLL mods
  • Check for overlay-related crashes
Survival Mechanics
  • Advanced food mechanics with spoilage
  • Camping system
  • Challenges and achievements
  • Cooking overhaul
  • Dynamic weather effects
  • Temperature system
Combat System
  • Better Locational Damage
  • Stagger and bleeding effects
  • Stealth killmoves
  • Weapon and armor balance
  • Realistic recoil and spread
  • Advanced AI behavior
